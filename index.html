This is Tandoori UI

Meet the Cowork.io's UI library !
You will find here all the UI elements you should use to build Cowork.io's views.
We aim to replace all the legacy SASS code with this library.
In the meantime, all of the Tandoori UI css classes will be prefixed with tuiv2_ so you can easely identify new and old classes in the code.
Please feel free to get in touch with front team to discuss about any improvement and/or create an issue.

Grid :

Just like Twitter's Bootstrap, Tandoori UI has 12 columns.

We use the new flex CSS3 property. Use a tuiv2_flex-col--{screen breakpoint}-{size} inside a tuiv2_flex-row.
{screen breakpoint} refers to any of the 3 min-width we are using :

  • sm : 768px
  • md : 992px
  • lg : 1200px

{size} refers to the number of columns you want your element to use.
You may use sizes between 1 and 12.
